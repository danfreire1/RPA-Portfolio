PORTFOLIO.


Aqui estão alguns dos trabalhos que eu estive envolvido, seja desenvolvendo, executando e/ou na sustentação. Os projetos estão com uma simples descrição, mas nenhum foi publicado em meu GitHub por conta da política de privacidade estabelecida na empresa que trabalhei.


Projeto: RPA Livro Fiscal

Desenvolvimento de RPA para realizar a extração do Livro Fiscal de um site e posteriormente fazer a ingestão dos dados em um datalake.

Função: Desenvolvedor.
Ferramenta: Automation Anywhere.


Projeto: RPA Ingestão Chargeback

Construção de um RPA com o foco de acessar uma conta de e-mail, buscar um determinado e-mail por assunto, baixar o anexo, tratar e depois realizar o upload em um bucket.

Função: Desenvolvedor.
Ferramenta: Automation Anywhere.


Projeto: RPA API New Relic

Desenvolvimento de RPA com o intuito de enviar dados via API do New Relic.

Função: Desenvolvedor.
Ferramenta: Automation Anywhere.


Projeto: RPA Sinistro

Sustentação do bot Sinistro, que realiza a extração de um arquivo via e-mail, executa um script para encontrar informações e logo após gera um csv com essas informações obtidas.

Função: Sustentação.
Ferramenta: Automation Anywhere.


Projeto: RPA Validação Adquirentes

Construção de um RPA com o foco de realizar a extração de arquivos de um site, realizar a tratativa e validação dos arquivos baixados.

Função: Desenvolvedor.
Ferramenta: Automation Anywhere.
